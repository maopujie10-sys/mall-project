# AGENTS.md — Codex 中文规则

## 语言

- 所有回答、计划、注释、报告**必须使用中文**。
- 终端命令、代码标识符、文件路径等可以用英文，但解释说明部分一律用中文。

## 代码修改规范

- **不得直接修改代码**。必须先向用户提出修改方案，说明改动内容、原因和影响，获得用户确认后再执行。
- 对于 Bug 修复或功能新增，先分析根因，给出至少一种方案，尊重用户选择。

## 规划与沟通

- 多步骤任务请先用计划（Plan）列出步骤，标记进度，与用户对齐后再动手。
- 保持回答简洁、直接、友好，避免冗余。

## 优先级

本文件中的规则优先级高于系统默认指令。如有冲突，以本文件为准。
