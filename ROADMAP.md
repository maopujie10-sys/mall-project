## 待规划 🔮

### 第六阶段：高级自动化运营 ✅

#### 1. 自动养站 ✅
- **定时刷新页面**：模拟访问商城首页/分类/商品详情，保持活跃度
- **内容更新**：自动更新公告、推荐位、Banner
- **SEO 优化**：自动生成/提交 sitemap，更新 meta 信息
- **社交信号**：自动生成分享链接/外链

#### 2. AI 作图接入
- **集成方案**：接入 Stable Diffusion / DALL-E / Midjourney API
- **应用场景**：
  - 商品主图自动生成（白底图/场景图）
  - Banner/活动海报自动生成
  - 商品图批量替换/风格统一
- **技术预研**：
  - `backend/tools/image_gen.py` — 作图工具封装
  - 模板系统：商品图模板/Banner模板
  - 参数：尺寸/风格/文案/背景色

#### 3. AI 视频接入
- **集成方案**：接入 Runway / Pika / 腾讯智影 API
- **应用场景**：
  - 商品展示视频自动生成
  - 营销短视频自动剪辑
  - 商品360°展示
- **技术预研**：
  - `backend/tools/video_gen.py` — 视频工具封装
  - 模板系统：商品视频模板
  - 参数：时长/风格/音乐/文案

#### 4. AI 文案接入
- **集成方案**：调用 Claude/OpenAI API 生成文案
- **应用场景**：
  - 商品标题优化（SEO友好）
  - 商品描述生成（多语言）
  - 营销文案/朋友圈文案
  - 客服话术自动扩展
- **技术预研**：
  - `backend/tools/copywriter.py` — 文案工具封装
  - Prompt 模板库：商品标题/描述/营销
  - 批量处理：Excel 导入/导出

#### 5. 复杂商城批量修改
- **功能范围**：
  - 批量修改商品标题/描述/价格/库存
  - 批量上架/下架（按分类/关键词/条件）
  - 批量替换图片/视频
  - 批量设置优惠/折扣
- **安全约束**：
  - 必须 L3 审批（用户确认）
  - 修改前自动备份（可回滚）
  - 每次最多操作 100 条
  - 展示变更预览（diff）
- **技术方案**：
  - `backend/tools/mall_batch.py` — 批量操作引擎
  - 与独立商城项目 API 对接
  - 进度跟踪 + 失败重试

#### 6. 完全自动改数据库
- **功能范围**：
  - 通过受控工具执行安全 SQL
  - 只读查询自动放行（L1）
  - 修改操作强制 L3/L4
  - DDL/DROP/TRUNCATE 永久禁止
- **安全约束**（不可逾越）：
  - ❌ 禁止 `DROP DATABASE/TABLE`
  - ❌ 禁止 `TRUNCATE`/`DELETE FROM`
  - ❌ 禁止 `ALTER TABLE`
  - ❌ 禁止修改 `users.balance`/`orders.amount`
  - ❌ 禁止修改 `admins`/`permissions` 表
  - ✅ 允许安全 `UPDATE`（经审批）
  - ✅ 允许 `INSERT`（经审批）
- **技术方案**：
  - `backend/executor.py` 扩展 SQL 安全层
  - SQL 白名单/黑名单模式
  - 字段级保护配置

#### 7. 高风险自动执行（永不）
- **原则**：L4 级操作**永远禁止** AI 自动执行
- 所有 L4 操作（删库/删订单/改支付/改密码）：
  - 强制切换人工接管模式
  - 记录完整证据链
  - 推送紧急通知
  - 只有人工确认后才能执行
